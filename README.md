# AI Search Chat Fixer

Пользовательский скрипт (UserScript) для оптимизации UX и исправления логики ввода в веб-интерфейсе ИИ-поиска Google.

## 🎯 Решаемая проблема

Встроенный чат Google по умолчанию отправляет сообщение по нажатию клавиши `Enter`. Для ИТ-специалистов, QA-инженеров и аналитиков, привыкших писать длинные, логически структурированные промпты с абзацами и списками, это приводит к двум болям:
1. Случайная отправка недописанного текста при попытке сделать перенос строки.
2. Игнорирование или склеивание строк веб-интерфейсом при чтении истории.

## ✨ Ключевые фичи

Скрипт полностью перехватывает управление клавиатурой в поле ввода чата и реализует предсказуемое поведение в стиле современных IDE:
- **`Enter`**: Нативно переносит строку (делает разрыв `<br>` через прямую DOM-инъекцию). Сообщение НЕ отправляется.
- **`Shift + Enter`**: Дублирует логику переноса строки (стандартное поведение браузера).
- **`Alt + Enter`**: Для однозначности взаимодействия с интерфейсом чата, обрабатывается аналогично `Enter`.
- **`Ctrl + Enter`**: Осознанно отправляет сформированный текст, используя встроенный скрытый триггер Google.

## 🛡️ Безопасность и изоляция

Скрипт разработан с учетом строгих правил безопасности:
- Область действия ограничена исключительно страницей результатов поиска (`https://google.com*`).
- Не внедряется на главную страницу Google, в Картинки, Карты или сторонние сайты.
- Не собирает, не кэширует и не отправляет ваши данные.

## 🚀 Установка и запуск

1. Установите один из менеджеров пользовательских скриптов для вашего браузера:
   - [Tampermonkey](https://tampermonkey.net) (рекомендуется)
   - [Violentmonkey](https://violentmonkey.github.io)
2. Откройте панель управления расширения и нажмите **"Создать новый скрипт"**.
3. Скопируйте содержимое файла `src/ai_search_chat_fixer.js` из этого репозитория и вставьте в редактор расширения.
4. Нажмите **Ctrl + S** для сохранения.
5. Перезагрузите страницу с поиском Google.
